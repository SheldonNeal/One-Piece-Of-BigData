##常见问题  
+ error: You have not concluded your merge (MERGE_HEAD exists).  
1.解决方法一：保留本地的更改,中止合并->重新合并->重新拉取亲测有效 
``` 
$ git merge --abort
$ git reset --merge
$ git pull
```
2.解决方法二：舍弃本地代码,远端版本覆盖本地版本(慎重)
```
$ git fetch --all
$ git reset --hard origin/master
$ git fetch
```
